Where do I put my links?
Edit the buttons (left side) in every page and change their href:
  • Telegram: set to your channel/group or bot (e.g., https://t.me/yourchannel).
  • 𝕏: set to your profile on X (e.g., https://x.com/YourHandle).
  • Whitepaper & Playbook: link to PDFs or site pages.
  • Airdrop: link to your form or dApp.
  • CA: link to your contract address page (e.g., Solscan/Raydium).
How do I make a button a different color?
In assets/styles.css, find --btn-bg and --btn-hover. Change them. Or, target a specific button by label using nth-child, example: 
.sidebar .navlink:nth-child(3) {{ /* the third link (𝕏) */
  background: linear-gradient(135deg, #444, #111);
}}
How do I make a button use an image?
Add class img and inline style with a background image: 
<a class="navlink img" href="playbook.html" style="background-image:url('assets/btn-texture.jpg')">Playbook</a>
GitHub Pages 101 (why did my page show just code?)
Make sure your repository has:
  1. A file named index.html in the root (top folder).
  2. Go to Settings → Pages, select Deploy from a branch, choose main and /(root).
  3. Wait 1–2 minutes, then open the Pages URL it shows you.
